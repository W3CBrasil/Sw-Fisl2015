Web Semântica

RDF

  • Resource Description Framework
  • É um modelo para representação de dados
  • não é um formato
  • Define uma maneira de explicar relações

RDF

  • Triplas!
  • Entidade => atributo => valor
  • Sujeito => predicado => objeto
  • entity - evento - FISL

    • http://softwarelivre.org/fisl16 - http://purl.org/dc/dcmitype/Event - FISL 2015

    Um conjunto de triplas é um RDF graph
